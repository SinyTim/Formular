Каталог



Программа "Формуляр" позволяет решать основные задачи торговли:

  • управление ассортиментом.
  • пополнение товарных запасов, обеспечение стабильности ассортимента.
  • контроль за товарными запасами и оборачиваемостью.
  • контроль за сроками годности и сертификатами.
  • контроль за поставщиками.
  • расчеты с поставщиками.
  • централизованное управление ценовой политикой.
  • предоставление оперативных данных для принятия решения, аналитическая отчетность.
  • предоставление отчетности для бухгалтерии и экспорт в бухгалтерскую программу 1С.
  • оформление собственно производства, получение типовых форм документов.
  • проведение акций и анализ их результатов.
  • сокращение численности персонала.
  • обеспечение конфиденциальности и безопасности.

Система автоматизации «Ретмастер» состоит из нескольких отдельных программных модулей.



Базовый блок

Это модуль выполняющий основные операции и служащий обязательной базой для других блоков.

Основные функции:

  • поступление/отпуск товара.
  • списание товара.
  • инвентаризация.
  • ведение справочников системы.
  • поддержка специализированного оборудования.
  • ценообразование.
  • контроль за товарными запасами на этапе заказа и оприходования.
  • отчет о превышении запасов и рекомендации к возврату.
  • отчет по оборачиваемости.
  • остатки товаров.
  • движение по товарам(закупка/продажа/история розничных цен).
  • бухгалтерские отчеты (ежедневные, ежемесячная/годовая статистика).
  • контроль за сертификатами и сроками годности.
  • экспорт в «1С бухгалтерию».
  • управление «пакетом фоновых заданий» (автоматический сбор чеков с касс/запуск отчетов на ночь).
  • статистические отчеты по чекам(средний чек/повременной отчет/по суммам чеков/по озвратам/отменам/коррекциям).

Блок «КАССА»

Обеспечивает взаимодействие FormulaR с POS терминалами. Данный блок позволяет подключать кассы, работающие на фронт-офисах Maxi-Shop, SetRetail, UniPos, БелМаг, СуперМаг УКМ-4, PS POS2000, НТС-ФронтОфис (MiniMARKET).

Основные функции:

  • ведение файла товаров.
  • переоценка и изменение данных на POS терминалах.
  • сбор/передача данных с касс/на кассы.

Блок «ЗАКАЗ»

  • автоматизированный заказ товаров.
  • контроль за ассортиментом и ценами поставщика.
  • заказ недостающих товаров.
  • заказ по результатам продаж.

Блок «ЦЕНТРАЛИЗОВАННЫЙ ЗАКАЗ»

Он позволяет осуществлять заказы товаров для всех магазинов из головного офиса.
